🟢 3. What is if Statement?
🧠 Definition

if is used to execute code only when a condition is true.

💻 Syntax
if (condition) {
  // code runs if condition is true
}
💻 Example 1 (Basic)
let age = 18;

if (age >= 18) {
  console.log("You can vote");
}
🧠 Explain
Condition → age >= 18
If true → runs code
If false → nothing happens
🟢 4. if-else (Two Conditions)
🧠 Definition

Used when we have two outcomes (true or false)

💻 Syntax
if (condition) {
  // true block
} else {
  // false block
}
💻 Example (Pass/Fail)
let marks = 30;

if (marks >= 35) {
  console.log("Pass");
} else {
  console.log("Fail");
}
🌍 Real-Life Use Case

Student result system ✔️

🟢 5. else if (Multiple Conditions)
🧠 Definition

Used when we have multiple conditions

💻 Syntax
if (condition1) {
  
} else if (condition2) {
  
} else {
  
}
💻 Example (Grades System)
let marks = 75;

if (marks >= 90) {
  console.log("A Grade");
} else if (marks >= 70) {
  console.log("B Grade");
} else if (marks >= 50) {
  console.log("C Grade");
} else {
  console.log("Fail");
}
🧠 Explain Flow
Checks one by one
First true condition runs
Others are skipped
🟢 6. Real-Life Example (Login System)
let password = "1234";

if (password === "1234") {
  console.log("Login Success");
} else {
  console.log("Wrong Password");
}
🟢 7. Real-Life Example (Even/Odd)
let num = 7;

if (num % 2 === 0) {
  console.log("Even");
} else {
  console.log("Odd");
}
🟢 8. Nested if (Basic Intro)
💻 Example
let age = 20;
let hasID = true;

if (age >= 18) {
  if (hasID) {
    console.log("Allowed");
  } else {
    console.log("ID Required");
  }
} else {
  console.log("Not Allowed");
}
